# نادي الجسر الرياضي (Al-Jisr Sports Club)

أكاديمية متكاملة لتطوير لاعبي كرة القدم وفق أعلى المعايير العالمية.

## عن المشروع
موقع إلكتروني تعريفي لنادي الجسر الرياضي، يبرز خدمات النادي، البرامج التدريبية، والعضويات المتاحة. يتميز الموقع بتصميم عصري (Dark Mode) وتجربة مستخدم سلسة تتضمن سلايدر تفاعلي وتأثيرات بصرية متقدمة.

## التقنيات المستخدمة
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- Google Fonts (Cairo)

## الميزات
- تصميم متجاوب (Responsive Design) متوافق مع كافة الأجهزة.
- سلايدر تفاعلي لعرض مميزات النادي.
- تأثير جزيئات الخلفية (Background Particles) لإضافة لمسة تقنية.
- تبديل بين الوضع الداكن والفاتح (Dark/Light Mode).

## طريقة التشغيل
ببساطة قم بفتح ملف `index.html` في أي متصفح حديث.
